# PSIB2024Q2-TPI-Grupo7-Ballester-Rodriguez-Rosado
Sistema computacional de apoyo para el análisis automático de imágenes microscópicas de frotis sanguíneo.

En este trabajo se desarrolló un sistema computacional de apoyo para el análisis de imágenes microscópicas de frotis sanguíneo, orientado a la clasificación automática de imágenes compatibles con patrón anémico o sano. Para ello, se utilizó el dataset AneRBC, específicamente el subconjunto AneRBC-II, debido a que sus imágenes presentan mayor aumento y menor cantidad de eritrocitos por campo, lo que permite reducir el costo computacional del procesamiento. El pipeline implementado incluyó etapas de preprocesamiento, segmentación de eritrocitos, extracción de características celulares, agregación de parámetros a nivel de imagen, construcción de un modelo de clasificación binaria basado en centroides normalizados y validación con imágenes etiquetadas. 
La segmentación se realizó mediante umbralización de Otsu, operaciones morfológicas, transformada de distancia y algoritmo Watershed, con el objetivo de separar los eritrocitos del fondo y dividir células en contacto. A partir de las regiones segmentadas se extrajeron características morfológicas, de intensidad, color, textura e índice de palidez central. Posteriormente, dichas características fueron agregadas estadísticamente para construir un descriptor global por imagen. El clasificador se basó en la distancia euclídea a los centroides de las clases Anemic y Healthy en un espacio normalizado. Finalmente, el sistema fue evaluado sobre un conjunto de validación independiente y se implementó un módulo de análisis para imágenes nuevas, capaz de mostrar la clasificación automática, las métricas principales y las visualizaciones asociadas al proceso de segmentación.
